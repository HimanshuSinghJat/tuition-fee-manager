# Tuition Fee Manager

A simple web application to manage student tuition fees.

---

## Features

* Add students with:

  * Name
  * Class
  * Monthly fee
  * Start month and end month

* Dashboard:

  * Track monthly payments (Paid / Pending)
  * Future months are automatically blocked
  * Advance payment handling
  * Total paid and total due calculation
  * Highlights students with multiple pending months

* Utilities:

  * Search student by name
  * Sort by name or class
  * Edit student after session ends
  * Delete student

---

## How It Works

* Data is stored in the browser using localStorage
* No backend or database is used
* Works completely offline
* Data is device-specific

---

## Important Notes

* Data is saved only in your browser
* Do not clear browser data, or all records will be lost
* Data will not sync across devices

---

## Usage

1. Open the app
2. Add students from the main page
3. Go to dashboard to manage fees
4. Click buttons to mark payments as Paid or Pending

---

## Tech Stack

* HTML
* CSS
* JavaScript
* localStorage

---

## Purpose

This project is designed for small tuition classes and non-technical users who need a simple fee tracking system.
